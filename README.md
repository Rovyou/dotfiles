# My i3 Dotfiles

* **Dependencies:**
    <img src="https://github.com/Rovyou/dotfile-i3wm/blob/main/images/desktop.gif?raw=true" alt="Rice Showcase" align="right" width="500px">
    * Zsh
    * Nvim
    * Oh My Zsh (optional)
    * Picom
    * Alacritty
    * Rofi
    * Polybar
    * Ranger
  
* **Installation:**
       Clone this repository to your home directory:

       
       git clone https://github.com/Rovyou/dotfile-i3wm.git
       
* **Customization:**
    * Edit the configuration files in the `config` directory.
