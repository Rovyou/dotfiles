# My i3 Dotfiles
**Description:**
A collection of my personal dotfiles to customize my development environment.

**Information:**
* **Purpose:** To streamline my workflow and create a consistent development experience across different machines.
* **Dependencies:**
    <img src="https://github.com/Keyitdev/screenshots/blob/master/dotfiles/v3/screenshots/1.png?raw=true" alt="Rice Showcase" align="right" width="200px">
    * Zsh
    * Nvim
    * Oh My Zsh (optional)
    * Picom
    * Alacritty
    * Rofi
    * Polybar
    * Ranger
    * 
* **Installation:**
       Clone this repository to your home directory:

       
       git clone https://github.com/Rovyou/dotfile-i3wm.git
       
* **Customization:**
    * Edit the configuration files in the `config` directory.
